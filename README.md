# Dvp Images
Dvp - programm for adding tegs to images and searching by selected tegs. 
## Описание
Dvp Images – приложение для добавления к изображениям тегов и последующего поиска по ним.
Основная сфера использования – поиск мемов, относящихся к классу «для важных переговоров», чтобы при необходимости быстро найти и отправить мем в беседу/диалог в какой-либо соц. сети или мессенджере. Также неплохо подходит для сортировки изображений, находящихся на жестком диске. 
## Внешний вид
Слева располагается область выбора тегов для поиска
В центре – таблица с изображениями, содержащая id, название (такое же, как в проводнике) и расположение.
Справа находится окно управления выбранным изображением: 
- данные о нём (название, расположение, разрешение и dpi)
- предпросмотр
- таблица с тегами, которые привязаны к этому           изображению 
 - небольшой интерфейс работы с этими тегами
Сверху меню с дополнительными функциями: 
- Открытие окна настроек
- Кнопка «Refresh» для обновления списка изображений и тегов.
- Открытие окна продвинутого поиска
- «About» - просто about
# Основы использования
### Загрузка изображений
1. Для начала нам необходимо добавить папку с картинками, чтобы добавить их в базу данных.
Откроем «Settings», кликнув на соответствующих пункт в меню сверху.
Появилось окно настроек. Теперь есть несколько способов добавить папку:
1)	Просто указать расположение папки в поле для текста.
(Например, «C:\Users\admin\Pictures»)
И нажать кнопку «Add Folder»
2)	Нажать «Browse» -> выбрать папку. Расположение в любом случае впишется в поле. –> «Add Folder»
    Папка добавлена –> выберем её кликом левой кнопкой мыши
    Нажимаем «Download Images»
    Если сделать всё правильно, то появится заветная надпись об успешном добавлении.
    Added images: 7 – количество добавленных изображений
(Теперь, при каждом добавлении новых файлов в эту папку на жестком диске, мы можем воспользоваться той же функцией «Download Images»,     и будут загружены только картинки, которые отсутствуют в программе.)
2.Изображения добавлены, можем закрыть окно настроек.
Не забываем нажать кнопку “Refresh” в верхнем меню, чтобы обновить таблицы.
### Добавление Тегов
1. Выбираем первую картинку в центральной таблице двойным кликом лкм.
2. Пишем в поле ввода (справа от кнопки "Add teg") необходимый тег. 
3.Нажимаем «Add Teg»
Хорошо, тег добавлен и появился в небольшой таблице слева. Будем называть её списком с тегами. Но каждый раз вводить по одному тегу не очень удобно, поэтому предусмотрена следующая функция
**В поле ввода можно ввести сразу несколько тегов, перечисленных через** ***запятую и пробел***. **Тогда программа разделит эти теги и добавит по отдельности.**
*«Clear entry after adding»* - по стандарту включена, очищает поле ввода тегов после их добавления, можно отключить. Удобно использовать, если нужно к нескольким пикчам добавить одни и те же теги – просто отключаем её.
**Функция «Choose tegs»**
Эта функция вставляет в поле ввода тегов те теги, которые находятся в листе выбора.

В таблице «Tegs» слева появились теги. В ней будут отображаться все теги, которые есть у изображений. Будем называть эту таблицу «таблица тегов». Справа от каждого тега указано количество изображений, к которым он привязан. (Увы, работает не всегда точно…) 
## Поиск по тегам
В таблице тегов двойным кликом теги добавляются в таблицу, которая находится ниже, будем называть её список выбора. При этом тег пропадёт из таблицы тегов.
Также, если сделать двойной клик по тегу в списке выбора, тег исчезнет из выбранных и снова появится в таблице.
Функция «Deselect All» возвращает все выделенные теги обратно в таблицу тегов.
Теперь, когда необходимые теги добавлены в лист выбора, нужно нажать на кнопку «Search».
Можно проводить поиск сразу по нескольким тегам, в результате будут отображены изображения, к которым привязаны **все** теги, которые находятся в списке выбора
## Продвинутый поиск
Чтобы открыть окно продвинутого поиска, нажмём на кнопку «Search…» в верхнем меню. Здесь поиск происходит с помощью совмещения функций, реализованных в поле добавления тегов и в обычном поиске.  
Например, вводим тег «Night», нажимаем ОК, и мы получаем тот же результат, что и при обычном поиске. 
Можно вводить несколько тегов через пробел, это будет эквивалентно обычному поиску с этими тегами в листе выбора. 
Есть одно важное различие: при обычном поиске изображение, с тегом «Night_Town» не попадет в поиск по тегу «Night». А при Advanced Search попадёт, так как проверяется даже части тега, можно ввести лишь одну букву, и поиск выдаст все изображения, в которых хотя бы в одном теге есть эта буква. Регистр тоже не учитывается.
## Дополнительные функции
1. "Тайтл" программы изменяется, отображая дополнительные данные о поиске. Показывает теги, по которым проводился поиск и количество найденных изображений.

2. Функция «Открыть в программе для просмотра изображений»
  Двойной клик на окно предпросмотра открывает картинку в программе, выбранной в качестве стандартного просмотра фотографий.
  
3. Удаление изображения из базы данных
  Справа сверху в главном окне находится кнопка «Delete», которая удаляет из базы данных выбранное изображение. Обращаем внимание на то,   что пикча останется на диске, но не будет отображаться в программе.
  (Если в настройках снова загрузить изображения из папки, то все удалённые пикчи снова попадут в базу данных)
  
4. Удаление тегов.
 Настройки – вкладка Tegs
 Двойной клик по тегу – появляется кнопка Delete
 При нажатии появляется предупреждение, что этот тег удалится у всех изображений. Нажимаем ДА или НЕТ. 
 
5. Изменение и удаление добавленных папок
Двойной клик по папке в настройках показывает дополнительные кнопки по управлению директориями. 
В поле ввода автоматически вписывается расположение папки. Его можно изменить и нажать кнопку Change, тогда у всех изображений с это директорией она изменится. По нажатию кнопки Delete появляется подтверждение на удаление, потом окно выбора: удалить также все изображения, занесенные в базу данных из этой папки или же оставить их. (Как и при обычном удалении изменения произойдут только в базе данных, файлы на диске останутся нетронутыми)
